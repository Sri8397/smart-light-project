# Server Setup Guide

1. Create a virtual environment:
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt

2. Set up MongoDB (local or remote).
3. Run the server:
   python main.py

4. API Endpoints:
   POST   /api/upload-video     - Upload video files
   GET    /api/logs             - View all events
   GET    /api/devices          - List connected devices
   POST   /api/control          - Send control command to RPi
